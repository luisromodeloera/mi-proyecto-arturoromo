🧭 Pasos para crear un nuevo repositorio en GitHub

  1. Inicia sesión en GitHub
    Ve a https://github.com e inicia sesión con tu cuenta.

  2. Crea un nuevo repositorio

    • Haz clic en el botón + en la esquina superior derecha.

    • Selecciona "New repository".

  3. Rellena la información del repositorio

    • Repository name: elige un nombre para tu repositorio (por ejemplo, mi-proyecto-python).

    • (Opcional) Description: escribe una breve descripción del proyecto.

    • Selecciona la opción "Public" o "Private" según tu preferencia.

  4. Inicializa el repositorio con contenido

    • Marca la opción "Add a README file".

    • Marca la opción ".gitignore" y elige Python de la lista de plantillas.

    • (Opcional) Puedes también seleccionar una licencia (por ejemplo, MIT).

  5. Crea el repositorio

    • Haz clic en el botón verde "Create repository".

Con estos pasos tendrás un repositorio en GitHub que contiene:

  • README.md: archivo inicial de documentación.

  • .gitignore: configurado para proyectos Python, ignorando archivos como __pycache__/, .pyc, env/, etc.

import pandas as pd
import plotly.express as px
import streamlit as st

# Título de la app
st.header("Análisis de anuncios de autos - Streamlit")

# Leer los datos
car_data = pd.read_csv('vehicles_us.csv')

# Botón para histograma
if st.button('Mostrar histograma de odómetro'):
    st.write('Histograma del odómetro')
    fig = px.histogram(car_data, x="odometer")
    st.plotly_chart(fig, use_container_width=True)

# Botón para gráfico de dispersión
if st.button('Mostrar gráfico de dispersión (odómetro vs precio)'):
    st.write('Gráfico de dispersión entre odómetro y precio')
    fig2 = px.scatter(car_data, x="odometer", y="price")
    st.plotly_chart(fig2, use_container_width=True)
